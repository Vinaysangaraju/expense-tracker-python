# Expense Tracker (Console-Based)

This is a simple console-based Expense Tracker application built using Python.

## Features
- Add expenses with amount, category, and notes
- View all added expenses
- Menu-driven interface
- Basic input validation

## Concepts Used
- Python OOPS (classes and objects)
- Constructors
- Composition
- Loops and conditional logic
- Basic exception handling

## How to Run
1. Clone the repository
2. Run the Python file:
   python expense_tracker.py
